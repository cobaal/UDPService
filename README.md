## Installing

	git clone https://github.com/cobaal/UDPService

## How to use

 - Compile
 
		cd UDPService
		make 

 - Receiver Execution
 
		 ./udpreceiver [IP Address] [Port #]

 - Receiver Termination

		Q

 - Sender Execution

		./udpsender [Destination IP] [Destination Port #]

 - Periodically Send Messages

		[t/s]macro
		msec : [msec]

 - Periodically Send Messages Termination

		exit

 - Sender Termination

		exit
